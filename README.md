#  Currículo Técnico & Engenharia de Performance — v1

Este repositório contém a arquitetura inicial e a interface responsiva do meu currículo profissional focado em desenvolvimento web e performance de conversão. 

>  **Nota de Desenvolvimento:** Este projeto representa a **Fase 1** de uma infraestrutura proprietária definitiva. O objetivo atual foi consolidar a identidade visual minimalista, aplicar conceitos de scannability e criar um código limpo, rápido e sem dependências pesadas.

---

##  Tecnologias Utilizadas

Para garantir máxima velocidade de carregamento e pontuação ideal de performance (Core Web Vitals), a estrutura foi desenvolvida de forma purista:
* **HTML5:** Estruturação semântica e organizada por blocos lógicos.
* **CSS3 Moderno:** Arquitetura baseada em variáveis nativas (`:root`), Flexbox e CSS Grid para responsividade total, eliminando a necessidade de frameworks (Bootstrap/Tailwind).
* **Google Fonts:** Integração das fontes *Space Grotesk* (identidade e títulos) e *JetBrains Mono* (dados e elementos técnicos).

---

##  Conceito Visual & UI Design

O design foi projetado sob uma estética industrial/developer escura, priorizando o contraste e a velocidade de leitura para o usuário final:
* **Foco em Scannability:** Remoção de blocos fechados e excesso de cards para criar uma leitura contínua orientada por linhas e divisores sutis.
* **Hierarquia de Cores:** Fundo sóbrio em tom de grafite escuro, textos de apoio em azul de alta fidelidade e destaque numérico estratégico em amarelo.
